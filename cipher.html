Playfair Cipher

Invented by Charles Wheatstone in 1854, the Playfair cipher encrypts pairs of letters (digraphs) using a 5×5 key matrix built from a keyword. It was the first practical digraph substitution cipher, widely used by British forces in WWI and WWII.

Build the 5×5 key matrix
Write the keyword (removing duplicates), then fill remaining cells with unused letters A–Z (I and J share a cell).
2
Prepare the message
Remove spaces. Split into digraphs (pairs). Insert X between identical pairs. Pad with X if length is odd.
3
Apply the three rules per digraph
Same row: shift each letter right (encrypt) or left (decrypt).
Same column: shift each letter down (encrypt) or up (decrypt).
Rectangle: swap to opposite corners in the same row.

Hill Cipher

Invented by Lester S. Hill in 1929, this cipher uses linear algebra — specifically matrix multiplication modulo 26 — to encrypt blocks of letters. It was the first cipher to work on blocks larger than 3 symbols at a time.

Monoalphabetic Cipher

One of the oldest cryptographic techniques — each letter of the alphabet is consistently mapped to a different fixed letter. The key is the substitution alphabet (26 letters rearranged). Can be keyword-based or fully random.
